# Mean-Variance-Analysis-Cryptocurrency
There's a bot that lets you to implement the Harry Markowitz portfolio analysis to the cryptocurrency market
Sustaiable eonly with Bybit, restricts some coins. 

To use:
1. download the .py file
2. open it with any IDE
3. insert there api_key and api_secret of your account
4. Run the code. You will see the interface in the terminal, run "calculate portfolio".insert necessary data. Then you will see a picture of efficient frontier -> choose a portfolio on it, close the picture, type the number of portfolio. Buy the portfolio by "buy" button.
5. To sell the portfolio, press "cash out".
6. To sell everything in your acccount to USDT, press "cash out to USDT".

The bot works only with spot pairs and USDT. 
The underlying theory - Harry Markowitz portfolio theory.
